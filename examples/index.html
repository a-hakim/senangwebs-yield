SenangWebs Yield (SWY)

Lightweight Data Visualization Library

Now using HTML, CSS & SVG for better versatility and responsive design instead of Canvas!

Key Features

  • ✓Fully responsive and scales beautifully on all screen sizes
  • ✓Easy to customize with CSS - no coding knowledge required
  • ✓Better accessibility - semantic HTML elements
  • ✓Zero dependencies - lightweight and fast
  • ✓SVG charts are crisp on any resolution (Retina ready)
  • ✓Dual API: HTML attributes or JavaScript methods
  • ✓Auto-initialization from HTML markup
  • ✓Interactive hover effects and animations

HTML API Examples

Define charts directly in HTML using data attributes - no JavaScript needed!

Bar Chart (HTML API)

<div data-swy>
  <div data-swy-type="bar-chart"
       data-swy-x-axis="Quarters"
       data-swy-y-axis="Revenue ($K)">
    <div data-swy-x-label="Q1"
         data-swy-y-value="150"
         data-swy-color="#ff6600"></div>
    ...
  </div>
</div>

Pie Chart (HTML API)

<div data-swy>
  <div data-swy-type="pie-chart"
       data-swy-total="360">
    <div data-swy-label="Web Design"
         data-swy-value="120"
         data-swy-color="#ff6600"></div>
    ...
  </div>
</div>

Line Chart (HTML API)

<div data-swy>
  <div data-swy-type="line-chart"
       data-swy-x-axis="Days"
       data-swy-y-axis="Active Users">
    <div data-swy-x-label="Mon"
         data-swy-y-value="50"></div>
    ...

</div>

JavaScript API Examples

Create charts programmatically with JavaScript for dynamic data

Bar Chart (JS API)

<div id="bar-chart-js"></div>
<script>
SWY.initBarChart({
  container: '#bar-chart-js',
  xAxis: 'Months',
  yAxis: 'Revenue',
  data: [
    { xLabel: 'Jan', yValue: 300, color: '#ff6600' },
    { xLabel: 'Feb', yValue: 400 },
    ...
  ]
});
</script>

Pie Chart (JS API)

<div id="pie-chart-js"></div>
<script>
SWY.initPieChart({
  container: '#pie-chart-js',
  total: 500,
  data: [
    { label: 'Service A', value: 200 },
    { label: 'Service B', value: 150 },
    ...
  ]
});
</script>

Line Chart (JS API)

<div id="line-chart-js"></div>
<script>
SWY.initLineChart({
  container: '#line-chart-js',
  xAxis: 'Weeks',
  yAxis: 'Sales',
  data: [
    { xLabel: 'Week 1', yValue: 100 },
    { xLabel: 'Week 2', yValue: 150 },
    ...
  ]
});
</script>

Quick Start

  • ✓Include swy.min.css and swy.min.js in your HTML
  • ✓Add data-swy attributes to your HTML elements
  • ✓Charts auto-initialize on page load
  • ✓Or use the JavaScript API for dynamic data
  • ✓Customize colors, labels, and styling with CSS
  • ✓Charts automatically resize on window resize
